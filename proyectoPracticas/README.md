# Gestor de Tareas

Proyecto final realizado con React y Vite.

## Tecnologías usadas

- React
- JavaScript
- CSS
- LocalStorage

## Funciones

- Crear tareas
- Editar tareas
- Eliminar tareas
- Completar tareas
- Filtrar tareas
- Ordenar tareas
- Guardado automático en localStorage

## Ejecutar proyecto

Instalar dependencias:

npm install

Iniciar servidor:

npm run dev
